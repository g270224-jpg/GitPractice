# Day 003 - GitHub Workflow

Today I learned the basic GitHub workflow.

The main commands are:

1. git status - checks what changed
2. git add . - prepares files to be saved
3. git commit - saves the changes locally
4. git push - uploads the changes to GitHub

Git is the tool on my computer.
GitHub is the online place where my code is stored.

Basic workflow:

change file -> git status -> git add . -> git commit -m "message" -> git push
